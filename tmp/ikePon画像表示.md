## 3. 画面レイアウト（改定）

### 3.1 全体構成（メインウィンドウ上部）

最上部のレイアウトを以下のように変更し、左端に **PANICボタン**、右端に **FULLボタン** と **DISPLAYボタン** を対称に配置します。ボタンの大きさは、現在のPANICボタンと同じです。

```
┌──────────────────────────────────────────────────────────────────┐
│ メニューバー（ファイル / 表示 / 設定）                              │
├──────────┬──────────────────────────────────────────┬──────┬─────┤
│  PANIC   │ インフォメーションエリア（2行）          │ FULL │ DISP│
│  (赤)    │                                          │ (緑) │ (緑)│
├──────────┴──────────────────────────────────────────┴──────┴─────┤
│ ...（以下、BANKエリア・PADエリア・MIXERエリア）                     │

```

* **FULLボタン（右端から2番目）背景：暗めの緑。ショートカットキー `[0]`。
* **DISPLAYボタン（右端）背景：暗めの緑。ショートカットキー `[-]`。

#### ボタンの視覚的状態（トグル挙動）

* **DISPLAYボタン**
* **OFF（デフォルト）:** 消灯状態（グレー背景）。映像出力クローズ。
* **ON:** 枠と文字が **黄色** に点灯。指定されたディスプレイに動画ウィンドウ（または全画面）がスタンバイされ、動画再生を受け付ける状態になります。


* **FULLボタン**
* **OFF（デフォルト）:** 消灯状態（グレー背景）。別ウィンドウ画面で動画を表示します。
* **ON:** 点灯状態（緑背景）。動画ウィンドウを全画面表示にします。対象となるディスプレイは、ウィンドウが表示されていたディスプレイです。

## 5. 再生制御・動画ウィンドウ仕様（新規・改定）

### 5.1 動画ウィンドウ（Movie Window）の挙動

* **初期状態（DISPLAY: OFF / FULL: OFF）:**
* 縦横比 **16:9**（初期サイズ **1280x720**）の別画面ウィンドウとして表示されます。
* マウスドラッグによる自由な拡大縮小が可能（最小サイズは **320x180**）。
* ウィンドウの位置・サイズ（状態）はアプリ終了時に自動記憶され、次回起動時に完全に再現されます。


* **全画面（フルスクリーン）化のトリガー:**
* 動画ウィンドウの **ダブルクリック**、またはメイン画面の **FULLボタン押下（キーボード `[0]`）** により、現在ウィンドウが存在するディスプレイで全画面化します。
* 全画面表示中に再度ダブルクリック、またはFULLボタンを押下すると、記憶された元のウィンドウ状態に戻ります。



### 5.2 動画非再生時の表示（ムービー画面静止画）

* 動画が再生されていない（Idle）状態のとき、動画ウィンドウには設定された静止画ファイル（またはデフォルトの黒画面）が表示され、イベント中の壁紙露出（放送事故）を物理的に防ぎます。

### 5.3 連動するセーフティ挙動（改定）

* **手動でのB.O.（ブラックアウト）操作:**
本番中に映像だけを即座に消したい場合、操作パネルの **DISPLAYボタン** をクリック（または `[-]` キー）してOFFにするだけで、動画ウィンドウが即座に消去・非表示になります。
* **PANICボタン（ESCキー）との連動:**
* **PANIC 1回押し:** 音声と同様に、再生中の動画も指定のフェードアウトを実行します。
* **2回押し（即時停止）:** 全音声・動画の即時停止に加え、**DISPLAYボタンが自動的にOFF** になり、動画ウィンドウを強制クローズします。万が一メインモニターで全画面化してしまった場合も、ESC連打で確実に操作画面を救出できます。

## 8. 設定パラメーター & 設定画面（新規・改定）

### 8.1 「設定」ダイアログの追加項目

メニュー「設定 → 設定」ウィンドウの**一番上**に以下の項目を追加します。

* **ムービー画面静止画:**
* 選択されているファイルのフルパスを表示（テキストボックス等）。
* パッド詳細設定と同様に、エクスプローラーからドラッグ＆ドロップ（D&D）でのファイル登録に対応します。
* 未設定時は「`別ファイルに変更：ここへドラッグ＆ドロップ`」と案内テキストを表示。
* 行の右端に **「×」ボタン** を設置。押下すると登録ファイルがクリアされます。
* **仕様:** ファイルが未設定、またはファイルの読み込みに失敗した場合は、デフォルトとして **真っ黒な画面（Blackout）** を表示します。



### 8.2 設定ファイル（`settings.json`）の拡張

既存のパラメーターに以下が紐付け、または追加されます。

| プロパティ | デフォルト | 説明 |
| --- | --- | --- |
| MovieMode | Window | Window / FullScreen（FULLボタンと完全連動） |
| DisplayOutputActive | false | DISPLAYボタンのON/OFF状態（デフォルトはfalse） |
| MovieStandbyImagePath | "" | ムービー画面静止画のファイルパス（空なら黒画面） |
| MovieWindowRect | (自動) | ウィンドウ表示時の位置（X, Y）およびサイズ（Width, Height）の記憶用 |

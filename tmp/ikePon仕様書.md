# ikePon 仕様書

**現在バージョン: v1.0.49**
**最終更新: 2026-07-05（v1.0.49）**

---

## 1. システム概要

- **アプリ名:** ikePon（イベント用ポン出しアプリ）
- **プラットフォーム:** Windows 10/11 x64 専用、C# / WPF / .NET 9
- **オーディオエンジン:** NAudio 2.2.1
  - 出力優先順位: **WASAPI Exclusive（100ms+）** → WaveOutEvent（200ms/4buf）→ WASAPI Shared
  - WASAPI Exclusive は Windows オーディオミキサーを完全バイパス（ステレオ拡張等エフェクトを排除）
- **エンジン内部フォーマット:** 44100Hz ステレオ IEEE float32
- **コンセプト:** 現場での誤動作を防ぐ高信頼性・低レイテンシーの音響ポン出しシステム

---

## 2. プロジェクト構成

- **バンク数:** 8（A〜H）
- **パッド数:** 1バンクあたり 16（4×4）= 合計 128 ファイルまで登録可
- **カテゴリ:** BGM / SE / MOVIE の 3 種類
  - デフォルト: 上 2 行（パッド 0〜7）= BGM、下 2 行（パッド 8〜15）= SE
- **保存形式:** JSON プロジェクトファイル（.ikp）
- **設定ファイル:** `%APPDATA%\ikePon\settings.json`

## 3. 画面レイアウト

### 3.1 全体構成（単一ウィンドウ）

```
┌────────────────────────────────────────────────────────┐
│ メニューバー（ファイル / 設定）                           │
├──────────┬────────────────────────────────┬──────┬──────┤
│  PANIC   │ インフォメーションエリア（2行）  │ FULL │ DISP │
│  (赤)    │                                │ (緑) │ (緑) │
├──────────┴────────────────────────────────┴──────┴──────┤
│  PADエリア（4行×4列）  │  BANKエリア（4行 × 2列）       │
│                        ├──────────────────────────────  │
│                        │  MIXERエリア（4本フェーダー）   │
└────────────────────────┴───────────────────────────────┘
```

### 3.2 インフォメーション・パニックエリア

- タイトルバー: `ikePon v1.0.36 — project.ikp` 形式
- インフォメーション行: 操作ガイド・警告（バンク切り替え確認時はハイライト）
- バンク切り替え確認中は右端に **YES / NO ボタン**を表示
- **PANIC ボタン**（赤・大型、幅 130px）: ESC キーまたはマウスクリック
- **FULL ボタン**（幅 130px）: 暗めの緑背景、ショートカット `[0]`。テキストは常に白。
  - OFF: 暗緑背景・暗緑枠
  - ON: 明緑背景・**黄色枠**（フルスクリーンアクティブ）。DISP が OFF でも有効にできる（次に DISP を ON にした時にフルスクリーンで開く）
- **DISP ボタン**（幅 130px）: 暗めの緑背景、ショートカット `[-]`。テキストは常に白。
  - OFF: 暗緑背景・暗緑枠
  - ON: 暗緑背景・**黄色枠**（映像出力アクティブ）

### 3.3 PAD エリア（左側）

- 4×4 の PadButton（UserControl）を UniformGrid で配置
- 各パッドの構成要素（上から下へ）:

```
┌─────────────────────────────────────────┐
│ [カテゴリ]              [AfterPlayback] │  ← 同じ高さに配置
│           ファイル名（中央）             │
│ [時間表示]                   [キーバッジ]│  ← 同じ高さに配置
│■■■■■■■■■■■■ デッドゾーン ■■■■■■■■■■■│  ← 誤タップ防止（10px）
│██████████████ プログレスバー ██████████│  ← タップでシーク（10px）
└─────────────────────────────────────────┘
```

#### カテゴリバッジ（左上）
- `BGM` / `SE` / `MOV` を表示
- **タップ（Idle 時のみ）**: MOV → BGM → SE → MOV の順で循環変更
- **再生中はタップ無効**（誤変更防止）

#### AfterPlayback バッジ（右上）
- 「最後まで再生した後」の挙動をアイコンで表示
- **タップで循環変更（再生中も変更可）**:
  - `■` そのまま終了（Stop）
  - `‖` 最終フレームで止める（FreezeLastFrame）― MOVIE カテゴリのみ選択可
  - `↻` ループ再生（Loop）― SE カテゴリでは選択不可
  - 現カテゴリで無効な選択肢はスキップ
- アイコン: Segoe MDL2 Assets フォントを使用

#### ファイル名（中央）
- カスタム表示名またはファイル名（拡張子除く）

#### 時間表示（左下、キーバッジと同行）
- フォント: Consolas 12pt SemiBold（等幅）
- **Idle 時**: 総時間のみ `1:23`
- **再生中・フェード中**: 現在時間/総時間 `0:45/1:23`
- ファイル未設定時は非表示

#### キーバッジ（右下）
- キーボードショートカット文字を表示（Consolas 16pt bold）

#### デッドゾーン（プログレスバーの直上、10px）
- クリックを吸収し、パッド本体のトリガーが発火しない
- 誤タップ防止のバッファゾーン

#### プログレスバー（最下端、10px）
- **再生中（Playing）:** バーが左から右へ伸びる（再生位置に連動）
- **フェードアウト中（FadingOut）:** フェード終了まで伸びたまま表示
- **停止後（Idle）:** バーを消去（幅 0）
- **タップでシーク**: 再生中にタップした位置へ即座に移動

#### パッド音量

- **音量**（旧称: パッド・ゲイン）: このパッドのみに適用する音量（100%=標準、0%〜500%）
  - ファイル読込時のみパッド上部中央に Consolas 12pt Bold で表示（例: `100%`）
  - クリック/ドラッグ（上下）/マウスホイールで変更。Shift で 10 単位
  - ドラッグ中はマウスキャプチャ。フォーカス時に % を除去して数値のみ編集可
  - **静止画ファイルの場合は非表示**（音声がないため）
- ファイル・ゲイン（全プロジェクト共通の素材ゲイン）は廃止。音量のみで調整する。

#### パッド カラー状態

| 状態 | ボーダー | テキスト | 背景（上部本体） | デッドゾーン（10px） |
|:--|:--|:--|:--|:--|
| Idle | グレー `#555` | 白 | グレー `#3C3C3C`（固定） | グレー `#3C3C3C` |
| Playing（BGM/MOV） | 黄色 `#FFD700` | 白 | グレー `#3C3C3C`（固定） | 青系 `#0E223A` |
| Playing（SE） | 黄色 `#FFD700` | 白 | グレー `#3C3C3C`（固定） | グレー `#3C3C3C` |
| FadingOut | 黄→グレーへ補間 | 白 | グレー `#3C3C3C`（固定） | グレー（即時） |
| Shift 押下中（再生時） | — | 白 | — | 青系 `#0E223A` |
| Ctrl 押下中（再生時） | — | 白 | — | 赤系 `#3A0E0E` |

※ 上部本体（ファイル名・バッジ・時間表示エリア）の背景色はユーザーカスタマイズ用に固定。状態変化はデッドゾーンとボーダーのみに反映。

### 3.4 BANK エリア（右上）

- **レイアウト:** 4 行 × 2 列の UniformGrid（左列: A/C/E/G、右列: B/D/F/H）
- 各ボタンの表示: バンク名（カスタム可）+ キーバッジ（右下、14pt bold）
- 選択中バンク: ボーダーハイライト（`#FFD700` 黄色）
- **右クリックメニュー:** 詳細設定（表示名・背景色を設定）/ 設定をコピー / 設定をペースト（16 パッド一括）
- **バンク背景色:** 詳細設定ダイアログで 8 色から選択可（デフォルト=ダークグレー）。横幅を広げて 8 色を 1 行に配置

### 3.5 MIXER エリア（右下）

- **4 本の縦フェーダー（VFaderControl）:** MOVIE / BGM / SE / MASTER
  - ラベル色: MOVIE=紫、BGM=青、SE=緑（カテゴリバッジと同色）、MASTER=グレー
- **スケール:** Logic Pro 準拠（−∞ 〜 +6dB）
  - 上端 = +6dB（ゲイン 1.9953）、75% 位置 = 0dB（ゲイン 1.0）、下端 = −∞
- **M1〜M4 メモリーボタン:** 各フェーダー下に 2×2 配置
  - クリック（記憶なし）: 現在値を登録
  - クリック（記憶あり）: 記憶値へ移動（0.5 秒）
  - Shift + クリック: ゆっくり移動（3.0 秒）
  - 右クリック: 登録メニュー（上書き確認あり）

### 4.3 修飾キー操作

| 操作 | 挙動 |
|:--|:--|
| キー単押し | 通常トリガー |
| Shift + キー | 該当パッドをロングフェードアウト（デフォルト 3.0 秒） |

| Y | バンク切り替え確認で YES |
| N | バンク切り替え確認で NO |

## 5. 再生制御

### 5.1 通常トリガー挙動

1. **同カテゴリの別パッドが再生中** → 旧パッドはショートフェード（0.5 秒）停止、新パッドを即再生
2. **同パッドを再押し（MOVIE/BGM のみ）** → フェードアウト停止（再起動しない）
3. **同パッドを再押し（SE）** → 最初から再再生
4. **アイドル** → 即座に再生開始
5. **MOVIE カテゴリ**: 音声再生と同時に動画ウィンドウへの映像再生も開始

### 5.2 インターロック

- 同一パッドは前回トリガーから 500ms 以内の入力を無視（連打防止）

### 5.3 PANIC

- **1 回押し:** 全パッドをロングフェードアウト（`LongFadeDuration` 秒）を**即座に**開始
  - フェード中: PANIC ボタンのボーダー・テキストが黄色になり、フェードに連動して元色（赤）に戻る
  - 動画も同時にフェードアウト
- **2 回連打（200ms 以内）:** 全パッドを即座に停止（`StopImmediate`）し、WASAPI を再初期化。**DISP を自動 OFF**（動画ウィンドウを強制クローズ）

### 5.4 バンク切り替え

- キーを押すと確認メッセージ表示（即切り替えではない）
- 確認中も現バンクのパッド操作は通常通り受付可
- 確定（YES）時: 旧バンクの全音をショートフェードで停止 → 新バンクに切り替え → バックグラウンドでロード

### 5.5 右クリックメニュー（パッド）

- **未再生パッド:** ファイルを読み込み / 設定をコピー / 設定をペースト / 詳細設定
- **再生中パッド:** フェードアウト / 即座に停止

### 5.6 パッド詳細設定ダイアログ

| 項目 | 説明 |
|:--|:--|
| カテゴリ | MOVIE / BGM / SE を切り替え |
| 表示名 | パッド上テキスト（デフォルト: ファイル名、拡張子除く） |
| 背景色 | パッド上部の背景色（8 色から選択、デフォルト=ダークグレー） |
| ファイルパス | ウィンドウへの D&D で再選択可能 |
| 音量 | このパッドのみに適用する音量（100%=標準、0%〜500%）。クリック/ドラッグ/ホイールで編集 |
| 再生開始位置 | 途中再生用タイムスタンプ（デフォルト 0:00.0） |
| 再生終了位置 | 途中終了用タイムスタンプ（デフォルト: ファイル末尾）。到達したらショートフェードで自動停止 |
| 最後まで再生した後 | Stop / FreezeLastFrame（MOVIE のみ） / Loop（SE 以外） |

- カテゴリ変更により「最後まで再生した後」の選択が無効になる場合は「そのまま終了」へ自動リセット

## 6. 動画ウィンドウ（Movie Window）

### 6.1 表示・非表示

- **DISP ボタン / `-` キー** でトグル
- **ON 時:** 動画ウィンドウが表示され、MOVIE カテゴリのパッド再生を受け付ける
- **OFF 時:** ウィンドウをクローズ（即座に映像が消える B.O. 操作にも利用可）

### 6.2 スタンバイ画像

- 動画が再生されていない（Idle）状態のとき、設定された静止画ファイルを表示
- ファイル未設定または読み込み失敗時は黒画面（Blackout）
- イベント中の壁紙露出（放送事故）を物理的に防ぐ

### 6.3 ウィンドウ操作

- **初期サイズ:** 1280×720（最小 320×180）
- **ダブルクリック** または **FULL ボタン / `0` キー:** 全画面化トグル
- **全画面解除:** 再度ダブルクリック or FULL ボタン → 記憶済みウィンドウ状態に復元
- 位置・サイズはアプリ終了時に自動記憶・次回復元

### 6.4 映像レンダリング

- **LibVLCSharp.WPF（VLC エンジン）** を使用
  - WASAPI Exclusive モードとの競合を回避（WMF MediaElement の 0xC00D11B1 エラーを根本解決）
  - `--no-audio` フラグで VLC 側の音声を無効化（ikePon の音声エンジンのみ使用）
  - VideoView は常時 Visible（HWND 確保のため）。スタンバイ時は StandbyLayer（WPF Grid）を最前面に表示
  - 映像再生開始時に StandbyLayer を Collapsed にして映像を露出
- **フェードアウト:** Window.Opacity を DispatcherTimer で段階的に変化させる方式
  - HWND ベースの VideoView も含むウィンドウ全体がフェード（WPF Airspace 問題を回避）
- **シーク:** `MediaPlayer.Position`（0.0〜1.0 の割合）を直接設定

### 6.5 「最後まで再生した後」の動作（MOVIE カテゴリ）

| 設定 | 動作 |
|:--|:--|
| Stop（そのまま終了） | 再生終了後スタンバイ画像に戻る |
| FreezeLastFrame（最終フレームで止める） | 最終フレームで一時停止・映像表示を継続 |
| Loop（ループ再生） | 開始位置に戻って繰り返し再生 |

- パッド右上の AfterPlayback バッジを**再生中でも**タップして変更可能、即時反映
- プログレスバーシーク時、音声と映像が**同時にシーク**する（SeekToFraction + MediaPlayer.Position）

## 7. 音声ロードシステム

### 7.1 完全プリロード（全ファイル共通）

- `AudioFileReader` → `WdlResamplingSampleProvider` → `MonoToStereoSampleProvider` の変換チェーンで一括デコード
- デコード済み float[] を RAM に保持
- 再生時は配列コピーのみ（GC 負荷なし）
- 10 分（600 秒）上限でデコードを打ち切るループガード付き

### 7.2 対応フォーマット

**音声:** `.mp3` / `.wav` / `.flac` / `.ogg` / `.aac` / `.m4a`  
**動画（音声+映像）:** `.mp4` / `.mov` / `.mkv` / `.avi` / `.wmv`  
**静止画（映像のみ）:** `.jpg` / `.jpeg` / `.png` / `.bmp` / `.gif` / `.webp` / `.tiff` / `.tif`

- m4a / 動画ファイル: `AudioFileReader` 内部で `MediaFoundationReader` を使用（Windows 標準コーデック）
- 動画ファイル: 音声 + **映像出力**に対応（LibVLCSharp による映像再生）
- 静止画ファイル: **映像出力のみ**（音声なし）。パッド上に音量・時間を表示しない
- 動画・静止画ファイルを読み込むと**カテゴリが自動的に MOV に設定**される（後から変更可）

### 7.3 ゲイン計算

```
最終出力 = 素材 × ファイルゲイン（FileGainDatabase） × パッドゲイン × カテゴリフェーダー × マスターフェーダー
```

- FileGainDatabase: `%APPDATA%\ikePon\filegain.json` に保存（ファイル名+サイズ+タイムスタンプ → ゲイン値のマップ）
- ゲイン上限: 4.0（内部クランプ）

### 7.4 プログレスバーシーク

- 再生中にパッド下部のプログレスバーをタップすると、タップ位置に対応する時間へ即時ジャンプ
- `PadAudioSource.SeekToFraction(float fraction)` でメモリ上の再生位置を直接変更

## 8. PAセパレートモード

「設定」ダイアログ最上部の「PAセパレートモード」プルダウンで ON/OFF 切り替え。

- **OFF（通常）:** 全カテゴリをステレオでそのままミックス
- **ON:** 全音声をモノダウンミックスして L/R に分配
  - **Lチャンネル:** BGM + MOVIE
  - **Rチャンネル:** SE

## 9. 設定パラメーター（AppSettings）

| プロパティ | デフォルト | 説明 |
|:--|:--|:--|
| PaSeparateMode | false | PA セパレートモード |

| FaderQuickMoveDuration | 0.5 秒 | メモリーボタン通常クリック時のフェーダー移動時間 |
| FaderSlowMoveDuration | 3.0 秒 | Shift+クリック時のフェーダー移動時間 |
| WasapiLatencyMs | 30 ms | WASAPI バッファサイズ |
| InterLockMs | 500 ms | 同一パッドの連打ブロック時間 |
| MovieMode | Window | Window / FullScreen（FULL ボタンと連動） |
| DisplayOutputActive | false | DISP ボタン ON/OFF 状態 |
| MovieStandbyImagePath | "" | ムービー画面静止画のファイルパス（空なら黒画面） |
| MovieWindowX/Y/Width/Height | (自動) | ムービーウィンドウの位置・サイズ（終了時に自動記憶） |
| WindowWidth/Height | (自動) | メインウィンドウのサイズ |

### 設定ダイアログの項目

- PAセパレートモード（最上部）
- フェード時間（ショート / ロング）
- WASAPI レイテンシー
- プリロード閾値
- **ムービー画面静止画**: D&D または参照ボタンでファイル登録、×ボタンでクリア

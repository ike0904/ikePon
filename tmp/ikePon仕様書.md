# ikePon 仕様書

**現在バージョン: v1.0.6**
**最終更新: 2026-06-29**

---

## 1. システム概要

- **アプリ名:** ikePon（イベント用ポン出しアプリ）
- **プラットフォーム:** Windows 10/11 x64 専用、C# / WPF / .NET 9
- **オーディオエンジン:** NAudio 2.2.1、WASAPI 共有モード（レイテンシ 30ms デフォルト）、失敗時は WaveOut フォールバック
- **エンジン内部フォーマット:** 44100Hz ステレオ IEEE float32
- **コンセプト:** 現場での誤動作を防ぐ高信頼性・低レイテンシーの音響ポン出しシステム

---

## 2. プロジェクト構成

- **バンク数:** 8（A〜H）
- **パッド数:** 1バンクあたり 16（4×4）= 合計 128 ファイルまで登録可
- **カテゴリ:** BGM / SE / MOVIE の 3 種類
  - デフォルト: 上 2 行（パッド 0〜7）= BGM、下 2 行（パッド 8〜15）= SE
- **保存形式:** JSON プロジェクトファイル（.json）
- **設定ファイル:** `%APPDATA%\ikePon\settings.json`

---

## 3. 画面レイアウト

### 3.1 全体構成（単一ウィンドウ）

```
┌──────────────────────────────────────────────────┐
│ メニューバー（ファイル / 表示 / 設定）              │
├──────────────────────────────────────────────────┤
│ インフォメーションエリア（2行）  │ PANIC ボタン（赤）│
├─────────────────────┬────────────────────────────┤
│                     │ BANKエリア（4行 × 2列）       │
│  PADエリア（4行×4列）│                             │
│                     ├────────────────────────────┤
│                     │ MIXERエリア（4本フェーダー） │
└─────────────────────┴────────────────────────────┘
```

### 3.2 インフォメーション・パニックエリア

- 1行目: アプリ状態・タイトル表示（例: `ikePon v1.0.6 — project.json`）
- 2行目: 操作ガイド・警告（バンク切り替え確認時はハイライト表示）
- バンク切り替え確認中は 2 行目右端に **YES / NO ボタン**を表示
- PANIC ボタン（赤・大型、幅 130px）: ESC キーまたはマウスクリック

### 3.3 PAD エリア（左側）

- 4×4 の PadButton（UserControl）を UniformGrid で配置
- 各パッド内の表示要素:
  - カテゴリバッジ（BGM / SE / MOV）左上
  - ファイル名（中央、折り返しあり）
  - キーバッジ（右下、フォント 14pt bold）
  - プログレスバー（下端の細い帯）

#### パッド カラー状態

| 状態 | ボーダー | テキスト | キーバッジ枠 | 背景 |
|:--|:--|:--|:--|:--|
| Idle | グレー `#555` | グレー `#CCC` | グレー `#4A4A4A` | カテゴリ色 |
| Playing | 黄色 `#FFD700` | 黄色 `#FFD700` | `#AA8800` | グレー `#3C3C3C` |
| FadingOut | 黄→グレーに補間 | 黄→グレーに補間 | 黄→グレーに補間 | グレー |
| Shift 押下中 | — | — | — | 青系 `#0E223A` |
| Ctrl 押下中 | — | — | — | 赤系 `#3A0E0E` |

- FadingOut 中のボーダー/テキスト/バッジは `FadeGain`（1.0→0.0）に応じて黄色→通常色へ線形補間

### 3.4 BANK エリア（右上）

- **レイアウト:** 4 行 × 2 列の UniformGrid（左列: A/C/E/G、右列: B/D/F/H）
- 各ボタンの表示: バンク名（カスタム可）+ キーバッジ（右下）
- キーバッジ フォント: 14pt bold
- 選択中バンク: 青ハイライト（背景 `#1C446E`、ボーダー `#3A9FFF`）
- **右クリックメニュー:** 名前を変更 / 設定をコピー / 設定をペースト
  - コピー: 16 パッド分の設定を一括クリップボードへ
  - ペースト: クリップボードから 16 パッド分を一括上書き

### 3.5 MIXER エリア（右下）

- **4 本の縦フェーダー（VFaderControl）:** MOVIE / BGM / SE / MASTER
- 各フェーダーの構成:
  - 縦スライダー（目盛り付き、範囲 0.0〜1.26）
  - ラベル（MOVIE / BGM / SE / MASTER）
  - **M1〜M4 メモリーボタン:** 2×2 配置（高さ 32px × 幅 1/2、フォント 11pt）
- 記憶あり: 点灯表示（`#3A7FC1` 系）、記憶なし: 消灯（`#2E2E2E`）
- ボリューム上限: 1.5（+3.5dB 程度のブースト可）

## 4. キーボード操作

### 4.1 パッドキー（4×4）

```
1  2  3  4
Q  W  E  R
A  S  D  F
Z  X  C  V
```

### 4.2 バンクキー（4行 × 2列）

```
7  8   ← バンク A, B
U  I   ← バンク C, D
J  K   ← バンク E, F
M  ,   ← バンク G, H  ※カンマ（OemComma）
```

### 4.3 修飾キー操作

| 操作 | 挙動 |
|:--|:--|
| キー単押し | 通常トリガー |
| Shift + キー | 該当パッドをロングフェードアウト（デフォルト 3.0 秒） |
| Ctrl + キー | 該当パッドをショートフェードアウト（デフォルト 0.5 秒）で停止 |
| ESC | PANIC（1 回目: フェードアウト、2 回目: 即座に停止） |
| Y | バンク切り替え確認ダイアログで YES |
| N | バンク切り替え確認ダイアログで NO |

## 5. 再生制御

### 5.1 通常トリガー挙動

1. 同カテゴリの**別パッドが再生中**の場合 → ショートフェード（0.5 秒）で停止後、新パッド再生
2. **同パッドが再生中**の場合 → ロングフェード（3.0 秒）開始
3. **アイドル**の場合 → 即座に再生開始

### 5.2 インターロック

- 同一パッドは前回トリガーから 500ms 以内の入力を無視（連打防止）

### 5.3 PANIC

- **1 回押し:** 全パッドをロングフェードアウト（`LongFadeDuration` 秒）
- **2 回連打:** 全パッドを `StopImmediate()`（サンプル単位で即停止）
- 1 回押しで即フェード開始 → 2 回目が来た瞬間に残りをハードカット（遅延なし）

### 5.4 バンク切り替え

- キーを押すと確認ダイアログ表示（即切り替えではない）
- **確認中も現バンクのパッド操作は通常通り受付可**
- 確定（YES）時: 旧バンクの全音をショートフェードで停止 → 新バンクに切り替え → バックグラウンドでロード

### 5.5 右クリックメニュー（パッド）

- **未再生パッド:** ファイルを読み込み / 設定をコピー / 設定をペースト / ゲイン調整
- **再生中パッド:** フェードアウト / 即座に停止

## 6. 音声ロードシステム

### 6.1 プリロード（≤ 閾値秒、デフォルト 10 秒）

- `AudioFileReader` → `WdlResamplingSampleProvider` → `MonoToStereoSampleProvider` の変換チェーンで一括デコード
- デコード済み float[] を RAM に保持
- 再生時は配列コピーのみ（GC 負荷なし）

### 6.2 ストリーミング（> 閾値秒）

- `AudioFileReader` を `_reader`（seek 用）として保持
- `BuildStreamProvider(_reader)` で変換プロバイダを生成して `_streamProvider` に保持
  - `WdlResamplingSampleProvider`（サンプルレート変換: 例 48000Hz→44100Hz）
  - `MonoToStereoSampleProvider` / `StereoToMonoSampleProvider`（チャンネル変換）
- 再生は `_streamProvider.Read()` を毎コールバックで呼ぶ
- seek（再トリガー）後は `BuildStreamProvider` を再生成してリサンプラー内部状態をリセット

### 6.3 対応フォーマット

`.mp3` / `.wav` / `.flac` / `.ogg` / `.aac` / `.m4a` / `.mp4` / `.mov` / `.mkv` / `.avi` / `.wmv`

- m4a: `AudioFileReader` が内部で `MediaFoundationReader` を使用（Windows 標準コーデック）
- 動画ファイル: 現状は音声のみ再生（MOVIE カテゴリに登録可能）

### 6.4 ゲイン計算

```
最終出力 = 素材 × ファイルゲイン（FileGainDatabase） × パッドゲイン × カテゴリフェーダー × マスターフェーダー
```

- FileGainDatabase: `%APPDATA%\ikePon\filegain.json` に保存（パス→ゲイン値のマップ）
- ボリューム上限: 1.5（0.0〜1.5 でクランプ）

## 7. PAセパレートモード

メニュー「表示 → PAセパレートモード」でトグル。

- **OFF（通常）:** 全カテゴリをステレオでそのままミックス
- **ON:** 全音声をモノダウンミックスして L/R に分配
  - **Lチャンネル:** BGM + MOVIE
  - **Rチャンネル:** SE

## 8. 設定パラメーター（AppSettings）

| プロパティ | デフォルト | 説明 |
|:--|:--|:--|
| ShortFadeDuration | 0.5 秒 | Ctrl 停止・同カテゴリ排他停止・バンク切り替え時のフェード |
| LongFadeDuration | 3.0 秒 | Shift フェード・同パッド 2 回押し・PANIC 1 回押し |
| FaderQuickMoveDuration | 0.5 秒 | メモリーボタン通常クリック時のフェーダー移動時間 |
| FaderSlowMoveDuration | 3.0 秒 | Shift+クリック時のフェーダー移動時間 |
| WasapiLatencyMs | 30 ms | WASAPI バッファサイズ |
| PreloadThresholdSeconds | 10 秒 | これ以下はプリロード、超えたらストリーミング |
| InterLockMs | 500 ms | 同一パッドの連打ブロック時間 |
| PaSeparateMode | false | PA セパレートモード |
| MovieMode | Window | AudioOnly / Window / FullScreen（FullScreen は未実装） |
| MovieMonitorIndex | 1 | 全画面モード時のモニタ番号 |

## 9. クラス・ファイル構成

```
src/ikePon/
├── MainWindow.xaml / .cs        メインウィンドウ・UI 制御
├── Audio/
│   ├── AudioEngine.cs           ISampleProvider ルート、128 ソース管理
│   ├── PadAudioSource.cs        1パッド分の音声ソース（プリロード/ストリーミング自動切替）
│   ├── FadeProcessor.cs         フェードアウト処理（struct）
│   └── PadPlayState.cs          enum: Idle / Playing / FadingOut
├── Controller/
│   ├── PlaybackController.cs    パッドトリガー・排他制御・インターロック
│   ├── BankManager.cs           バンク切り替え確認フロー
│   ├── PanicController.cs       1回/2回押し判定
│   └── KeyboardMapper.cs        キー→パッド/バンクインデックスのマッピング
├── Model/
│   ├── ProjectData.cs           プロジェクト全体（バンク配列 + 選択中バンク）
│   ├── BankData.cs              1バンク（PadSettings[16] + BankLabel）
│   ├── PadSettings.cs           1パッド（FilePath / Category / PadGain）

│   └── FileGainDatabase.cs      ファイルゲイン DB（filegain.json）
└── UI/
    ├── Controls/
    │   ├── PadButton.xaml/.cs   パッドボタン UI
    │   └── VFaderControl.xaml/.cs  縦フェーダー UI（M1-M4含む）
    └── Dialogs/
        └── SettingsDialog.xaml/.cs  設定ダイアログ
```

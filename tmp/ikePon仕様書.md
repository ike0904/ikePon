# ikePon 仕様書

**現在バージョン: v1.0.69**
**最終更新: 2026-07-06（v1.0.69）**

---

## 1. システム概要

- **アプリ名:** ikePon（イベント用ポン出しアプリ）
- **プラットフォーム:** Windows 10/11 x64 専用、C# / WPF / .NET 9
- **オーディオエンジン:** NAudio 2.2.1
  - 出力優先順位: **WASAPI Shared**（Windows ミキサー経由、他アプリと共存可） → WaveOutEvent（200ms/4buf） → WASAPI Exclusive（最終フォールバック）
  - WASAPI Shared 使用時: Windows のミックスフォーマット（48000Hz 等）を検出し、WdlResamplingSampleProvider で自動リサンプリング
- **エンジン内部フォーマット:** 44100Hz ステレオ IEEE float32
- **コンセプト:** 現場での誤動作を防ぐ高信頼性・低レイテンシーの音響ポン出しシステム

---

## 2. プロジェクト構成

- **バンク数:** 8（A〜H）
- **パッド数:** 1バンクあたり 16（4×4）= 合計 128 ファイルまで登録可
- **カテゴリ:** BGM / SE / MOVIE の 3 種類
  - デフォルト: 上 2 行（パッド 0〜7）= BGM、下 2 行（パッド 8〜15）= SE
- **保存形式:** JSON プロジェクトファイル（.ikp）
- **設定ファイル:** `%APPDATA%\ikePon\settings.json`

## 3. 画面レイアウト

### 3.1 全体構成（単一ウィンドウ）

```
┌──────────────────────────────────────────────────────────────┐
│ メニューバー（ファイル / 設定）                                 │
├──────────┬──────────┬──────────┬─────────────────┬─────────┬─────────┤
│  ALL CUT │ ALL FADE │  PAUSE   │ インフォメーション │FULL SCR │DISPLAY  │
│  (赤)    │  (青)    │  (緑)    │    エリア         │ (グレー)│(グレー) │
├──────────┴──────────┴──────────┴─────────────────┴─────────┴─────────┤
│  PADエリア（4行×4列）     │  BANKエリア（4行 × 2列）                 │
│                           ├────────────────────────────────────────── │
│                           │  MIXERエリア（4本フェーダー）             │
└───────────────────────────┴──────────────────────────────────────────┘
```

### 3.2 インフォメーション・パニックエリア

- タイトルバー: `ikePon v1.0.69 — project.ikp` 形式（アクセントなし）
- インフォメーション行: 操作ガイド・警告（バンク切り替え確認時はハイライト）
- バンク切り替え確認中は右端に **YES / NO ボタン**を表示

#### 5つのアクションボタン（全て高さ 44px・幅 128px で統一）

**ALL CUT**（赤・左端）
- ESC キーまたはマウスクリック
- 1 回押し: 全パッドをロングフェードアウト（LongFadeDuration 秒）+ 動画も同時フェード。ボーダーが黄色にアニメーション
- 2 回連打（200ms 以内）: 全パッドを即座に停止し、WASAPI を再初期化

**ALL FADE**（青・左から 2 番目）
- Space キーまたはマウスクリック
- 全再生中パッドをロングフェードアウト（LongFadeDuration 秒）
- 再生中のパッドがない場合は何もしない（視覚的変化なし）

**PAUSE**（緑・左から 3 番目）
- Enter キーまたはマウスクリック
- MOVIE / BGM カテゴリの全パッドを一時停止。SE は対象外
- 一時停止中: ボーダーが黄色に変化
- 再び押すと全パッドを再開
- 一時停止中に対象パッドの再生が別の方法で終了した場合は自動的に PAUSE 状態を解除

**FULL SCR**（グレー・右端から 2 番目）
- `0` キーまたはマウスクリック
- 動画ウィンドウの全画面/ウィンドウモードをトグル
- DISPLAY が OFF でも有効にできる（次回 DISPLAY を ON にした時にフルスクリーンで開く）

**DISPLAY**（グレー・右端）
- `-` キーまたはマウスクリック
- 動画ウィンドウを表示/非表示トグル

### 3.3 PAD エリア（左側）

- 4×4 の PadButton（UserControl）を UniformGrid で配置
- 各パッドの構成要素（上から下へ）:

```
┌────────────────────────────────────────────────┐
│ [カテゴリ]    [音量]      [AfterPlayback バッジ] │  ← 上部エリア（デッドゾーン）
│           ファイル名（中央）                      │
│ [時間表示]                        [キーバッジ]   │  ← 同じ高さに配置
│■■■■■■■■■■■■■■ デッドゾーン ■■■■■■■■■■■■■■│  ← 誤タップ防止（10px）
│████████████████ プログレスバー █████████████████│  ← タップでシーク（10px）
└────────────────────────────────────────────────┘
```

#### カテゴリバッジ（左上）
- `BGM` / `SE` / `MOV` を表示
- **タップ（Idle 時のみ）**: MOV → BGM → SE → MOV の順で循環変更
- **再生中はタップ無効**（誤変更防止）

#### 音量表示（中央上部）
- ファイル読込時のみ表示（例: `100%`）
- クリック/ドラッグ（上下）/マウスホイールで変更。Shift で 10 単位
- ドラッグ中はリアルタイムで音量を反映（確定前でも即座に適用）
- フォーカス時に % を除去して数値のみ編集可（Enter/ESC で確定/キャンセル）

#### AfterPlayback バッジ（右上）
- 「最後まで再生した後」の挙動をアイコンで表示
- **タップで循環変更（再生中も変更可）**:
  - `■` そのまま終了（Stop）
  - `‖` 最終フレームで止める（FreezeLastFrame）― MOVIE カテゴリのみ選択可
  - `↻` ループ再生（Loop）― SE カテゴリでは選択不可
  - 現カテゴリで無効な選択肢はスキップ

#### ファイル名（中央）
- カスタム表示名またはファイル名（拡張子除く）

#### 時間表示（左下、キーバッジと同行）
- フォント: Consolas 12pt（等幅）
- **Idle 時**: 開始位置/総時間 `0:00.0/1:23.5`（開始位置が 0 の場合は `0:00.0/1:23.5`）
- **再生中・フェード中**: 現在時間/総時間 `0:45.2/1:23.5`
- 0.1 秒単位表示（m:ss.t 形式）
- ファイル未設定時は非表示

#### キーバッジ（右下）
- キーボードショートカット文字を表示（Consolas 16pt bold）

#### 上部エリア（デッドゾーン）
- カテゴリ・音量・AfterPlayback バッジを含む上部エリア全体がデッドゾーン
- クリックを吸収し、パッド本体のトリガーが発火しない

#### デッドゾーン（プログレスバーの直上、10px）
- 再生中タップ設定（TapBehavior）に応じた色で表示:
  - SE カテゴリ: グレー（`#3C3C3C`）
  - CutOut（CTRL）: 赤系（`#3A0E0E`）
  - FadeOut（SHIFT）: 青系（`#0E223A`）
- 誤タップ防止のバッファゾーン（クリック無効）

#### プログレスバー（最下端、10px）
- **再生中（Playing）:** バーが左から右へ伸びる（再生位置に連動）
- **フェードアウト中（FadingOut）:** フェード進行に合わせてバーの色が黄色→グレーへ補間
- **停止後（Idle）:** バーを消去（幅 0）
- **タップでシーク**: 再生中にタップした位置へ即座に移動（音声・映像同時）

#### パッド カラー状態

| 状態 | ボーダー | ボーダー幅 | 背景（上部本体） | デッドゾーン |
|:--|:--|:--|:--|:--|
| Idle | グレー `#555` | 1.5px | カスタム色（デフォルト`#3C3C3C`） | TapBehavior 色 |
| Playing | 黄色 `#FFD700` | 2.5px | カスタム色（変化なし） | TapBehavior 色 |
| FadingOut | 黄→グレー補間 | 2.5px | カスタム色（変化なし） | グレー（即時） |

※ 上部本体の背景色はユーザーカスタマイズ用に固定。状態変化はデッドゾーンとボーダーのみ。

### 3.4 BANK エリア（右上）

- **レイアウト:** 4 行 × 2 列の UniformGrid（左列: A/B/C/D、右列: E/F/G/H）
- 各ボタンの表示: バンク名（カスタム可）+ キーバッジ（右下）
- 選択中バンク: ボーダーハイライト（`#FFD700` 黄色）
- **右クリックメニュー:** 詳細設定（表示名・背景色）/ 設定をコピー / 設定をペースト（16 パッド一括）
- **バンク背景色:** 詳細設定ダイアログで 8 色から選択可
- バンク切り替え: キーを押すと確認メッセージ表示（Y/N ボタンまたはキー）。確認中も現バンクのパッド操作は通常通り受付可

### 3.5 MIXER エリア（右下）

- **4 本の縦フェーダー（VFaderControl）:** MOVIE / BGM / SE / MASTER
  - ラベル色: MOVIE=紫、BGM=青、SE=緑（カテゴリバッジと同色）、MASTER=グレー
- **スケール:** 全 128 段階（0〜127）の線形スケール
  - 初期値 96 = ±0dB（ゲイン 1.0）、0 = 無音、127 = 最大（ゲイン約 1.32）
  - 目盛りは等間隔（96 の目盛り線は 2 倍太く白）
  - マウスホイール: 1 ステップ、Shift + ホイール: 8 ステップ
- **MU ボタン:** 各フェーダーの下部にミュートボタン。ON（赤）で対象カテゴリを無音化
- **M1〜M4 メモリーボタン:** 各フェーダー下に 2×2 配置
  - クリック（記憶なし）: 現在値を登録
  - クリック（記憶あり）: 記憶値へ移動（0.5 秒）
  - Shift + クリック: ゆっくり移動（3.0 秒）
  - 右クリック: 登録メニュー（上書き確認あり）

## 5. 再生制御

### 5.1 通常トリガー挙動（パッドキー単押し・クリック）

再生中タップ設定（TapBehavior）に応じて動作が異なる:

**FadeOut（デフォルト）**
1. **同カテゴリの別パッドが再生中** → 旧パッドはショートフェード（0.5 秒）停止、新パッドを即再生
2. **同パッドを再押し（MOVIE/BGM）** → ロングフェードアウト停止（再起動しない）
3. **同パッドを再押し（SE）** → 最初から再再生

**CutOut**
- 同パッドを再押し → 即座に停止（フェードなし）

**PauseResume（一時停止/再開）**
- 同パッドを再押し → 一時停止。もう一度押すと再開
- SE カテゴリでは選択不可

- **アイドル時の共通動作:** 全 TapBehavior で、アイドル状態のパッドをトリガーすると即座に再生開始
- **MOVIE カテゴリ**: 音声再生と同時に動画ウィンドウへの映像再生も開始
- **静止画パッド**: 音声なし。映像ウィンドウに画像を表示（音声系処理はスキップ）

### 5.2 インターロック

- 同一パッドは前回トリガーから 500ms 以内の入力を無視（連打防止）

### 5.3 ALL CUT（PANIC）

- **1 回押し:** 全パッドをロングフェードアウト（LongFadeDuration 秒）を**即座に**開始
  - フェード中: ALL CUT ボタンのボーダー・テキストが黄色になり、フェードに連動して赤に戻る
  - 動画も同時にフェードアウト
- **2 回連打（200ms 以内）:** 全パッドを即座に停止し、WASAPI を再初期化

### 5.4 ALL FADE

- 全再生中パッドをロングフェードアウト（LongFadeDuration 秒）
- 再生中のパッドがない場合は何もしない

### 5.5 PAUSE ALL

- MOVIE / BGM カテゴリの全再生中パッドを一時停止（SE は対象外）
- 一時停止中は PAUSE ボタンのボーダーが黄色に変化
- 再度押すと全パッドを再開（ボーダーが緑に戻る）
- 一時停止中に対象パッドの再生が終了した場合は自動的に PAUSE 状態を解除

### 5.6 バンク切り替え

- キーを押すと確認メッセージ表示（即切り替えではない）
- 確認中も現バンクのパッド操作は通常通り受付可
- 確定（YES）時: 旧バンクの全音をショートフェードで停止 → 新バンクに切り替え

### 5.7 右クリックメニュー（パッド）

**未再生パッド:**
- ファイルを読み込み
- 設定をコピー（ファイルパス / カテゴリ / 音量 / 表示名 / 背景色 / 開始・終了位置 / AfterPlayback / TapBehavior / LoopStart を全フィールドコピー）
- 設定をペースト（コピーした全フィールドを貼り付け）
- ファイルの割り当てをクリア（全設定をデフォルト値にリセット）
- 詳細設定...

**再生中パッド:**
- フェードアウト
- 即座に停止

**ファイルの割り当てをクリアの動作:**
- ファイルパス・表示名・背景色 → null（デフォルト）
- 音量 → 100%
- 開始位置 → 0:00.0
- 終了位置 → ファイル末尾（空欄）
- ループ開始位置 → 空欄
- 最後まで再生した後 → そのまま終了
- 再生中タップ → フェードアウト
- カテゴリ → パッド 0〜7（上 8 個）= BGM、パッド 8〜15（下 8 個）= SE

### 5.8 パッド詳細設定ダイアログ

| 項目 | 説明 |
|:--|:--|
| カテゴリ | MOVIE / BGM / SE を切り替え |
| 表示名 | パッド上テキスト（デフォルト: ファイル名、拡張子除く） |
| パッド背景色 | パッド上部の背景色（8 色から選択、デフォルト=ダークグレー） |
| ファイルパス | ウィンドウへの D&D で再選択可能（×ボタンでクリア） |
| 音量 | このパッドのみに適用する音量（100%=標準、0%〜500%）。クリック/ドラッグ/ホイールで編集 |
| 再生開始位置 | 途中再生用タイムスタンプ（デフォルト 0:00.0） |
| 再生終了位置 | 途中終了用タイムスタンプ（デフォルト: ファイル末尾=空欄） |
| 最後まで再生した後 | Stop / FreezeLastFrame（MOVIE のみ） / Loop（SE 以外） |
| ループ再生開始位置 | ループ時の折り返し先（デフォルト: 先頭=空欄）。Loop 選択時のみ有効 |
| 再生中タップ | フェードアウト / カットアウト / 一時停止・再開（SE では選択不可） |

- カテゴリ変更により選択不可になった項目は自動リセット
- 動画・静止画ファイルを D&D すると自動的に MOVIE カテゴリに設定

## 6. 動画ウィンドウ（Movie Window）

### 6.1 表示・非表示

- **DISPLAY ボタン / `-` キー** でトグル
- **ON 時:** 動画ウィンドウが表示され、映像出力を開始
- **OFF 時:** ウィンドウをクローズ（B.O. 操作にも利用可）

### 6.2 DISPLAY OFF 中の静止画対応

- DISPLAY が OFF の状態でも静止画パッドをトリガー可能
  - トリガーした画像の「表示待ち状態」を記録（内部で追跡）
  - DISPLAY を ON にすると記録された画像が即座に表示される
- 同じ画像パッドを再度トリガー → 表示待ちをキャンセル（DISPLAY ON 時に再表示されない）
- ALL CUT / ALL FADE 時は表示待ち状態も解除される

### 6.3 スタンバイ画像

- 映像が再生されていない状態のとき、設定された静止画ファイルを表示
- ファイル未設定または読み込み失敗時は黒画面（Blackout）

### 6.4 ウィンドウ操作

- **初期サイズ:** 1280×720（最小 320×180）
- **ダブルクリック** または **FULL SCR ボタン / `0` キー:** 全画面化トグル
- 位置・サイズはアプリ終了時に自動記憶・次回復元

### 6.5 映像レンダリング

- **LibVLCSharp.WPF（VLC エンジン）** を使用
  - `--no-audio` フラグで VLC 側の音声を無効化（ikePon の音声エンジンのみ使用）
  - VideoView は常時 Visible（HWND 確保のため）。スタンバイ時は StandbyLayer を最前面に表示
- **フェードアウト:** 黒い全画面 Topmost オーバーレイウィンドウの Opacity 0→1 フェードインで実現（VLC ForegroundWindow の上に確実に重ねる）
- **シーク:** `MediaPlayer.Position`（0.0〜1.0）を直接設定

### 6.6 「最後まで再生した後」の動作（MOVIE カテゴリ）

| 設定 | 動作 |
|:--|:--|
| Stop | 再生終了後スタンバイ画像に戻る |
| FreezeLastFrame | 最終フレームで一時停止・映像表示を継続 |
| Loop | 開始位置（またはループ開始位置）に戻って繰り返し再生 |

- パッド右上の AfterPlayback バッジを**再生中でも**タップして変更可能（即時反映）
- プログレスバーシーク時、音声と映像が**同時にシーク**（SeekToFraction + MediaPlayer.Position）

## 7. 音声ロードシステム

### 7.1 完全プリロード（全ファイル共通）

- `AudioFileReader` → `WdlResamplingSampleProvider` → `MonoToStereoSampleProvider` の変換チェーンで一括デコード
- デコード済み float[] を RAM に保持
- 再生時は配列コピーのみ（GC 負荷なし）
- 10 分（600 秒）上限でデコードを打ち切るループガード付き

### 7.2 対応フォーマット

**音声:** `.mp3` / `.wav` / `.flac` / `.ogg` / `.aac` / `.m4a`  
**動画（音声+映像）:** `.mp4` / `.mov` / `.mkv` / `.avi` / `.wmv`  
**静止画（映像のみ）:** `.jpg` / `.jpeg` / `.png` / `.bmp` / `.gif` / `.webp` / `.tiff` / `.tif`

- 動画・静止画ファイルを読み込むと**カテゴリが自動的に MOV に設定**される（後から変更可）
- 静止画ファイル: 音声なし。パッドの挙動は MOVIE カテゴリと同一（音量表示・時間表示・AfterPlayback バッジを含む）

### 7.3 ゲイン計算

```
最終出力 = 素材 × パッドゲイン × カテゴリフェーダー × マスターフェーダー
```

- パッドゲイン: 0%〜500%（100%=標準）
- ゲイン上限: 4.0（内部クランプ）

### 7.4 プログレスバーシーク

- 再生中にパッド下部のプログレスバーをタップすると、タップ位置に対応する時間へ即時ジャンプ
- 音声（PadAudioSource.SeekToFraction）と映像（MediaPlayer.Position）が同時にシーク

## 8. PAセパレートモード

「設定」ダイアログ最上部の「PAセパレートモード」プルダウンで ON/OFF 切り替え。

- **OFF（通常）:** 全カテゴリをステレオでそのままミックス
- **ON:** 全音声をモノダウンミックスして L/R に分配
  - **Lチャンネル:** BGM + MOVIE
  - **Rチャンネル:** SE

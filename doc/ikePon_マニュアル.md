# ikePon v1.0.72 ユーザーマニュアル

---

## 1. アプリケーション概要

**ikePon** は、イベント・ステージ現場向けのポン出しアプリです。最大 8 バンク × 16 パッド（計 128 ファイル）を登録し、BGM・SE・動画・静止画をキーボードまたはマウスで即座に再生できます。

| 項目 | 仕様 |
|:---|:---|
| 対応 OS | Windows 10/11 x64 |
| 開発環境 | C# / WPF / .NET 9 |
| オーディオエンジン | NAudio（WASAPI Shared モード） |
| 映像エンジン | LibVLCSharp |
| プロジェクトファイル | `.ikp`（JSON 形式） |
| 設定ファイル | `%APPDATA%\ikePon\settings.json` |

## 2. 画面構成

![メイン画面](pic/メイン画面.png)

画面は大きく **5 つのエリア** に分かれています。

| エリア | 場所 | 内容 |
|:---|:---|:---|
| メニューバー | 最上部 | ファイル操作・設定 |
| グローバルエリア | メニュー直下 | ALL CUT / ALL FADE / PAUSE / FULL SCR / DISPLAY の 5 ボタンとインフォメーション表示 |
| KEYPAD | 左側 | 4×4 パッドボタン（1 バンクあたり 16 個） |
| BANK | 右上 | バンク A〜H の切り替えボタン |
| MIXER | 右下 | MOVIE / BGM / SE / MASTER の 4 本フェーダー |

## 3. プロジェクトの管理

プロジェクトファイル（`.ikp`）に全パッドのファイル登録情報・バンク設定・ミキサー値を保存します。

| 操作 | メニュー | ショートカット |
|:---|:---|:---|
| 新規プロジェクト | ファイル → 新規プロジェクト | `Ctrl+N` |
| プロジェクトを開く | ファイル → 開く... | `Ctrl+O` |
| 上書き保存 | ファイル → 上書き保存 | `Ctrl+S` |
| 名前を付けて保存 | ファイル → 名前を付けて保存... | — |

**ヒント:** `.ikp` ファイルをメインウィンドウにドラッグ＆ドロップして開くこともできます。

## 4. パッドの設定と登録

### 4.1 ファイルの登録

音声・動画・静止画ファイルを **パッドにドラッグ＆ドロップ** するか、パッドを **右クリック → 詳細設定...** でファイルを選択します。

対応フォーマット：

| 種別 | 拡張子 |
|:---|:---|
| 音声 | `.mp3` `.wav` `.flac` `.ogg` `.aac` `.m4a` |
| 動画 | `.mp4` `.mov` `.mkv` `.avi` `.wmv` |
| 静止画 | `.jpg` `.jpeg` `.png` `.bmp` `.gif` `.webp` `.tiff` `.tif` |

動画・静止画ファイルを登録すると、カテゴリが自動的に **MOV** に設定されます。

### 4.2 パッド詳細設定ダイアログ

パッドを **右クリック → 詳細設定...** で開きます。

![パッド詳細設定](pic/パッド詳細設定.png)

| 項目 | 説明 |
|:---|:---|
| ファイル名（フルパス） | 登録されているファイルのパス。別ファイルへの変更は D&D で可。`×` でクリア |
| カテゴリ | `BGM` / `SE` / `MOV` を選択 |
| 表示名 | パッド上に表示する名前（空欄ならファイル名） |
| パッド背景色 | 8 色から選択（初期値：ダークグレー） |
| 音量 | このパッドのみの音量（100% = 標準、0〜500%） |
| 再生開始位置 | 途中から再生する場合の開始タイムスタンプ（例：`1:30.0`） |
| 再生終了位置 | 途中で終了する場合のタイムスタンプ（空欄 = ファイル末尾） |
| ループ再生開始位置 | ループ時の折り返し先（空欄 = 先頭から） |
| 最後まで再生した後 | `そのまま終了` / `最終フレームで停止`（MOV のみ） / `ループ`（SE 以外） |
| 再生中タップ | `フェードアウト` / `カットアウト` / `一時停止・再開`（SE は選択不可） |

### 4.3「最後まで再生した後」の動作

| 設定 | 動作 |
|:---|:---|
| そのまま終了 | 再生終了後、スタンバイ画像に戻る（BGM は無音になる） |
| 最終フレームで停止 | 最終フレームで静止して映像を表示し続ける（MOV のみ） |
| ループ | ループ再生開始位置（または先頭）に戻って繰り返す（SE 以外で有効） |

### 4.4 パッドボタンの見方

```
┌────────────────────────────────────────┐
│ [BGM]   100%                        [■] │  ← カテゴリ / 音量 / 再生後動作バッジ
│                                         │
│             ファイル名                   │  ← 中央に表示名
│                                         │
│  0:00.0/1:23.5                      [1] │  ← 時間表示 / キーバッジ
│■■■■■■■■■■■■■■■■ デッドゾーン ■■■■■■│  ← 10px（誤タップ防止）
│████████████████████ プログレスバー ████│  ← 10px（クリックでシーク）
└────────────────────────────────────────┘
```

#### パッドのカラー状態

| 状態 | ボーダー色 | 意味 |
|:---|:---|:---|
| 待機中（Idle） | グレー `#555` | 再生待機 |
| 再生中（Playing） | 黄色 `#FFD700` | 再生中 |
| フェードアウト中 | 黄色 → グレー補間 | フェードアウト中 |
| 行方不明（Missing） | 赤点滅 | ファイルが見つからない |

#### 右クリックメニュー

**待機中のパッド：**

| メニュー項目 | 動作 |
|:---|:---|
| 詳細設定... | パッド詳細設定ダイアログを開く |
| コピー | パッドの全設定をコピー |
| ペースト | コピーした設定を貼り付け |
| 削除 | パッドの設定を初期化 |

**再生中のパッド：**

| メニュー項目 | 動作 |
|:---|:---|
| フェードアウト | ロングフェードアウトして停止 |
| カットアウト | 即座に停止 |
| 一時停止／再開 | 一時停止・再開（SE カテゴリは無効） |

## 5. 再生操作

### 5.1 キーボードでの操作

パッドキーのレイアウトは以下のとおりです（キーボード上の位置に対応）。

```
 1   2   3   4    ← 上段（パッド 1〜4）
 Q   W   E   R    ← 2段目（パッド 5〜8）
 A   S   D   F    ← 3段目（パッド 9〜12）
 Z   X   C   V    ← 下段（パッド 13〜16）
```

| 操作 | 動作 |
|:---|:---|
| キーを押す | 通常再生（「再生中タップ」設定に従う） |

### 5.2 再生中タップの動作

パッド詳細設定の **「再生中タップ」** によって、再生中のパッドを再度押したときの挙動が変わります。

| 設定 | 同パッドを再押しした場合 |
|:---|:---|
| フェードアウト（デフォルト） | MOVIE/BGM：ロングフェードアウト停止。SE：最初から再生 |
| カットアウト | 即座に停止 |
| 一時停止・再開 | 一時停止。もう一度押すと再開（SE では使用不可） |

**同カテゴリの別パッドを再生した場合：** 旧パッドを即座に停止して新パッドを即再生します。

### 5.3 プログレスバーによるシーク

再生中にパッド下部の **プログレスバー（最下 10px）** をクリックすると、クリックした位置に即ジャンプします。音声と映像は同時にシークされます。

### 5.4 インターロック

同一パッドへの連打（500 ms 以内）は無視されます（設定で変更可）。

## 6. グローバル操作（ALL CUT / ALL FADE / PAUSE）

### 6.1 ALL CUT（赤ボタン / `ESC`）

全パッドを即座に一括停止する**パニックボタン**です。

| 操作 | 動作 |
|:---|:---|
| 1 回押し | 全パッドを即座に停止。ボタン枠が黄色にフラッシュ |

### 6.2 ALL FADE（青ボタン / `Space`）

再生中の全パッドをロングフェードアウトで停止します。再生中パッドがない場合は何も起きません。

### 6.3 PAUSE（緑ボタン / `Enter`）

MOVIE・BGM カテゴリの全再生中パッドを一時停止します（SE は対象外）。

| 状態 | ボタン枠色 |
|:---|:---|
| 通常（停止中） | 緑 |
| 一時停止中 | 黄色 |

もう一度押すと全パッドを再開します。一時停止中に該当パッドの再生が別の方法で終了した場合、PAUSE 状態は自動解除されます。

## 7. バンク（BANK）の使い方

### 7.1 バンクの構成

8 つのバンク（A〜H）がそれぞれ 16 個のパッドを持ちます。バンクキーのレイアウト：

```
 7   8    ← バンク A, E
 U   I    ← バンク B, F
 J   K    ← バンク C, G
 M   ,    ← バンク D, H  ※ カンマキー
```

現在選択中のバンクは **黄色の枠** でハイライトされます。

### 7.2 バンクの切り替え

- **再生中の音がない場合**：バンクキーを押すと即座に切り替わります（確認なし）。
- **再生中の音がある場合**：確認メッセージが表示されます。

```
インフォメーションエリアに確認メッセージ → [Y] 確定 / [N] キャンセル
```

確認中も**現バンクのパッド操作は通常どおり**受け付けます。確定すると旧バンクの全音声が即座に停止し、新バンクに切り替わります。

### 7.3 バンク詳細設定

バンクボタンを **右クリック → 詳細設定...** で開きます。

![バンク詳細設定](pic/バンク詳細設定.png)

| 項目 | 説明 |
|:---|:---|
| 表示名 | バンクボタンに表示する名前 |
| 背景色 | バンクボタンの背景色（8 色から選択） |

### 7.4 バンク右クリックメニュー

| メニュー項目 | 動作 |
|:---|:---|
| 詳細設定... | バンク詳細設定ダイアログを開く |
| コピー | バンク詳細設定およびバンク内 16 パッドの全設定をコピー |
| ペースト | バンク詳細設定およびバンク内 16 パッドの全設定を貼り付け |
| 削除 | バンク詳細設定およびバンク内の全パッドを初期化（確認あり） |

## 8. ミキサー（MIXER）の使い方

4 本の縦フェーダーで各カテゴリの音量を調整します。

| フェーダー | カテゴリ | ラベル色 |
|:---|:---|:---|
| MOVIE | 動画・静止画 | 紫 |
| BGM | BGM | 青 |
| SE | SE | 緑 |
| MASTER | 全体 | グレー |

最終出力音量 = パッド音量 × カテゴリフェーダー × MASTER フェーダー

### 8.1 フェーダー操作

| 操作 | 動作 |
|:---|:---|
| ドラッグ | フェーダー値を変更（0〜127） |
| マウスホイール | 1 ステップずつ変更 |
| `Shift` + ホイール | 8 ステップずつ変更 |

スケール：`0` = 無音、`96` = ±0 dB（標準）、`127` = 最大（約 +2.4 dB）

### 8.2 メモリーボタン（M1〜M3）

各フェーダーに M1〜M3 の 3 つのメモリーボタンがあります。

| 操作 | 動作 |
|:---|:---|
| クリック（未登録） | 現在のフェーダー値を登録 |
| クリック（登録済み） | 記憶した値へフェード移動 |
| 右クリック | フェード移動 / 即座に移動 / 再登録 / 削除 |

### 8.3 MU ボタン（ミュート）

各フェーダー下の **MU** ボタンを押すとカテゴリをミュート（黄色で ON 状態を表示）します。

## 9. 動画ウィンドウ（DISPLAY / FULL SCR）

### 9.1 DISPLAY（グレーボタン / `-` キー）

動画ウィンドウを表示・非表示でトグルします。

- DISPLAY が **OFF** の状態でも動画・静止画パッドをトリガーできます。DISPLAY を ON にすると最後にトリガーした映像が表示されます。

### 9.2 FULL SCR（グレーボタン / `0` キー）

動画ウィンドウの全画面/ウィンドウモードをトグルします。DISPLAY が OFF でも設定可能です（次回 ON 時に全画面で開きます）。

動画ウィンドウを**ダブルクリック**しても全画面/ウィンドウモードのトグルができます。

### 9.3 動画ウィンドウの操作

- ウィンドウの位置・サイズはアプリ終了時に自動保存され、次回起動時に復元されます。
- 最小サイズは 320×180 です。

### 9.4 スタンバイ画像

映像が再生されていない間、設定したスタンバイ画像を表示できます（設定 → スタンバイ画像）。未設定の場合は黒画面（ブラックアウト）になります。

## 10. スマートリロケート（自動ファイル再リンク）

プロジェクトファイル読み込み時に素材ファイルが見つからない（ドライブ変更やフォルダ移動など）場合、**自動的に素材を再検索して再リンク**します。

### 10.1 自動探索（フェーズ 1）

次の場所を順番に自動検索します。

1. プロジェクトファイルと同じフォルダおよびその全サブフォルダ
2. 前回ファイルを読み込んだフォルダ（外付けストレージのドライブレター変更に対応）

インフォメーションエリアに進捗が表示されます：

```
行方不明のファイルを自動再検索中... (2/5本)
```

自動探索が完了すると結果が表示されます：

```
3 本の行方不明ファイルを自動修復しました。
```

同名ファイルが複数見つかった場合、元のパスの**親フォルダ名**まで一致するものを優先します。

### 10.2 手動起点による一括解決（フェーズ 2）

自動探索で見つからないファイルがある場合、ダイアログが表示されます。**1 つのファイルを手動で指定**すると、そのフォルダを起点に残りのファイルも一括で再リンクを試みます。

### 10.3 行方不明ファイルの表示

最終的に再リンクできなかったパッドは **赤点滅のボーダー** と **[未リンク]バッジ** で表示され、再生は無効になります。

## 11. 設定ダイアログ（`Ctrl+E`）

![設定](pic/設定（全体）.png)

| 項目 | 説明 | デフォルト |
|:---|:---|:---|
| フェードアウト時間 | ALL FADE のフェード秒数 | 2.0 秒 |
| スタンバイ画像 | 動画ウィンドウの背景画像（D&D で設定、`×` でクリア） | なし（黒画面） |
| フェードイン時間 | スタンバイ画像の表示フェードイン時間 | 1.0 秒 |
| PAセパレートモード | ON にすると BGM+MOV → L チャンネル、SE → R チャンネルで出力 | OFF |
| 連打防止時間 | 同一パッドの連打を無視する時間 | 500 ms |
| WASAPI レイテンシ | オーディオバッファサイズ（変更後は再起動が必要） | 30 ms |
| プリロード上限 | プリロードする最大ファイル長（変更後は再起動が必要） | 10 秒 |

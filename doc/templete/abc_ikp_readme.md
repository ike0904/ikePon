# abc準拠イベント用テンプレート「abc.ikp」運用マニュアル

本項では、abc関連イベント専用に構成された専用プロジェクトファイル（テンプレート）である「abc.ikp」を導入し、現場でポン出し環境を即座に構築するための手順について解説します。

## 1. 概要

「abc.ikp」は、abcイベントで使用する音源や映像ファイルの配置スロット（パッド）、各種バンク設定、ミキサーの初期値などが事前に定義されたテンプレートファイル（JSON形式）です。
本アプリに搭載されている「スマートリロケート機能（自動ファイル再リンク）」を活用することで、環境（PCやドライブレター）が変わっても、指定のフォルダ内に配置するだけで一括して素材ファイルが有効化されます。

---

## 2. 導入・セットアップ手順

セットアップは以下の3ステップで完了します。

### ステップ 1：ファイルの配置

1. 使用するabcの楽曲・音声・映像素材がすべて格納されているフォルダを開きます。
2. そのフォルダのルート直下（第一階層）に、テンプレートファイル「`abc.ikp`」をコピー（または移動）して配置します。

### ステップ 2：プロジェクトの起動

ikePonを起動し、以下のいずれかの方法で「abc.ikp」を読み込みます。

* **メニューから開く**: メニューバーの `ファイル` ➔ `開く...`（またはショートカット `Ctrl + O`）から、ステップ1で配置した「abc.ikp」を選択します。
* **ドラッグ＆ドロップ**: 配置した「abc.ikp」ファイルを、ikePonのメインウィンドウに直接ドラッグ＆ドロップします。
* **ファイルをダブルクリック**: あらかじめOS側で拡張子 `.ikp` をikePonに関連付けしている場合は、ファイルをダブルクリックするだけでアプリの起動と同時に読み込まれます。

### ステップ 3：自動リロケートの実行（素材の自動紐付け）

プロジェクトが読み込まれると、ikePonの「スマートリロケート機能」が自動的に作動します。

1. `abc.ikp` が置かれたフォルダ、およびその配下にあるすべてのサブフォルダから、登録されているファイル名と一致する素材を最大30秒間（最大10万ファイル対象）自動で探索します。
2. 探索の進捗はインフォメーションエリアに「*行方不明のファイルを自動再検索中...*」とリアルタイム表示されます。
3. 探索が完了すると「*〇本の行方不明ファイルを自動修復しました。*」と通知され、すべてのパッドが待機状態（グレーのボーダー）に変わり、即座に出演・再生が可能な状態になります。

---

## 3. 運用上の注意点・トラブルシューティング

* **同名ファイルが複数存在する場合の挙動**
自動探索中に同一のファイル名が異なるフォルダから複数見つかった場合、テンプレートに記録されている「元のパスの親フォルダ名」と一致する素材が最優先でリンクされます。予期せぬ素材の誤リンクを防ぐため、素材フォルダ内は整理しておくことを推奨します。
* **赤点滅（Missing）表示になった場合**
自動探索でファイルがどうしても見つからなかったパッドは、ボーダーが赤く点滅し、[未リンク] バッジが表示されて再生機能がロックされます。
この場合は、表示される手動解決ダイアログに従って1つのファイルを指定するか、対象のパッドに対して正しい素材ファイルを再度ドラッグ＆ドロップして手動登録を行ってください。
